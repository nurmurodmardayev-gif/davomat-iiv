🔥 Firebase ulash kerak
Admin va xodimlar alohida qurilmalarda ishlashi uchun Firebase Firestore kerak. Quyidagi qadamlarni bajaring:
QADAM 1
Firebase loyiha yarating
1. console.firebase.google.com → "Create a project"
2. Loyiha nomini kiriting (masalan: davomat-tizim)
3. Google Analytics → "Continue" (yoki o'chiring)
QADAM 2
Web app qo'shing
Loyiha sahifasida → </> (Web) belgisini bosing
App nickname: davomat → Register app
firebaseConfig ni ko'chirib oling
QADAM 3
Firestore Database yoqing
Chap menuda → "Firestore Database" → "Create database"
→ Start in test mode → Next → Enable
QADAM 4
Firebase Config ni kiriting
Firebase console dan olingan config ma'lumotlarini kiriting:
💡 Firebase Spark rejimi (bepul): kuniga 50,000 o'qish, 20,000 yozish — kichik tashkilot uchun yetarli
